# Component Registry

The Devcn UI registry contains 48 components that you can add to your project.

## Available Components

A customizable accordion component.

```bash
npx devcn-ui add accordion
```

**Dependencies:** react, @radix-ui/react-accordion, lucide-react

## Installation

To install any component, use the Devcn UI CLI:

```bash
npx devcn-ui add [component-name]
```

You can also install multiple components at once:

```bash
npx devcn-ui add button card dialog
```
